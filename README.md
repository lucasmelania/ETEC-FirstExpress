# 🚀 ETEC - Primeiras Rotas com Express

Projeto desenvolvido em sala de aula com o objetivo de criar **rotas no Express** para apresentar diferentes módulos de usuário. Cada módulo corresponde a um exercício de lógica em JavaScript (Declaração, Operadores, Estruturas de Controle, Vetores etc.), e foi disponibilizado no navegador através de rotas específicas.

## 📌 Objetivo

Aprender a utilizar o **framework Express** para:
- Criar um servidor básico em Node.js  
- Importar funções de diferentes módulos JavaScript  
- Definir rotas (`app.get`) que retornam os resultados dos exercícios diretamente no navegador  

## ✅ Requisitos do Projeto

- [x] Instalar o **Express** via NPM  
- [x] Importar os módulos dos exercícios (A até H) no arquivo `index.js`  
- [x] Criar uma rota para cada exercício (`/exA`, `/exB`, … `/exH`)  
- [x] Exibir os resultados das funções no navegador
